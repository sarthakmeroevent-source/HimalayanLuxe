# HimalayanLuxe - Multi-Page Application

## Overview
Complete luxury event planning website with 5 fully designed pages, all maintaining the same luxurious aesthetic.

## Pages Created

### 1. Home Page (`/`)
- Hero section with video background and mute control
- Sticky Experience section with 4 philosophies (scroll-triggered content changes)
- Destinations carousel showcase
- Stacked services cards
- Call-to-action section
- Section pagination navigation (right side)

### 2. About Page (`/about`)
- Company story and philosophy
- Two-column grid layout with glass cards
- Statistics section (500+ events, 50+ countries, etc.)
- Team introduction
- Elegant animations and transitions

### 3. Portfolio Page (`/portfolio`)
- Filterable portfolio grid (All, Wedding, Destination, Corporate)
- 6 portfolio items with hover effects
- Category filter buttons
- Image overlays with project details
- Responsive masonry-style layout

### 4. Services Detail Page (`/services`)
- Detailed service descriptions
- Alternating layout (image left/right)
- 4 main services:
  - Wedding Planning
  - Destination Events
  - Design & Decor
  - Entertainment
- Feature lists for each service
- Call-to-action buttons

### 5. Contact Page (`/contact`)
- Full contact form with validation
- Contact information cards
- Office hours display
- Social media links
- Two-column layout (form + info)
- Glass morphism design

## Routing Structure
```
/ - Home Page
/about - About Page
/portfolio - Portfolio Page
/services - Services Detail Page
/contact - Contact Page
```

## Technical Stack
- React 18
- TypeScript
- React Router DOM
- Framer Motion
- Lenis (smooth scroll)
- Tailwind CSS
- Three.js (Silk background)

## Key Interactions
1. Smooth scroll with Lenis on all pages
2. Scroll-triggered content changes (Experience section on home)
3. Hover effects on portfolio items
4. Form validation on contact page
5. Category filtering on portfolio page
6. Sticky navigation with scroll detection

## Responsive Design
- Mobile-first approach
- Breakpoints: sm, md, lg
- Touch-friendly interactions
- Optimized layouts for all screen sizes
